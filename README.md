This Joomla component has been created for use by local Groups or Areas with Joomla 4/5 websites. The majority of its functionality is to provide a user-friendly, menu-driven interface to the powerful routines in the RamblersWebs Library.

Other utility views have been added potentially useful to local websites, including:
 - a document library
 - a file upload function
 - the ability to display articles from a different site
 - admin functions to display membership of security user groups
 - an admin function to display the database schema

Additionally, it has helper classes, standard fields and css classes that are available to, and widely used by, other components such as MailMan.
