How to Configure VS Code for ra-tools Development

Visual Studio Code (VS Code) is a powerful, free code editor that works excellently with the ra-tools project. This guide will help you set it up for optimal development.

Prerequisites

  • VS Code installed (download from code.visualstudio.com)
  • The ra-tools repository cloned to your local machine
  • Git installed and configured

Opening the Repository in VS Code

Method 1: Open from the command line

Navigate to the repository directory and open VS Code:

cd ~/git/ra-tools
code .

The dot (.) tells VS Code to open the current directory.

Method 2: Open from VS Code

  1. Launch VS Code
  2. Click File > Open Folder (or Ctrl+K Ctrl+O on Windows/Linux, Cmd+K Cmd+O on Mac)
  3. Navigate to the ra-tools directory
  4. Click "Select Folder" or "Open"

Recommended VS Code Extensions

Step 1: Open the Extensions view

Click the Extensions icon in the sidebar (or press Ctrl+Shift+X / Cmd+Shift+X)

Step 2: Install recommended extensions for PHP and Joomla development

Search for and install these extensions:

  • PHP Intelephense - PHP language support and IntelliSense
  • PHP Debug - Debug support for PHP
  • GitLens - Enhanced Git capabilities
  • EditorConfig - Maintain consistent coding styles
  • Better Comments - Improve your code comments
  • Prettier - Code formatter (optional but recommended)

Configure VS Code Settings for the Project

Step 3: Access workspace settings

  1. Press Ctrl+, (Windows/Linux) or Cmd+, (Mac) to open Settings
  2. Click on "Workspace" tab at the top
  3. This ensures your settings apply only to this project

Step 4: Configure file associations

Search for "files.associations" in settings and add:

{
"files.associations": {
"*.xml": "xml",
"*.php": "php"
}
}

Step 5: Set up PHP validation

If you have PHP installed locally, configure the path:

  1. Search for "php.validate.executablePath" in settings
  2. Set it to your PHP executable path (e.g., /usr/bin/php or C:\php\php.exe)

Configure Git Integration

Step 6: Verify Git integration

  1. Click the Source Control icon in the sidebar (or press Ctrl+Shift+G / Cmd+Shift+G)
  2. You should see the ra-tools repository and its current status
  3. If prompted, sign in to GitHub for enhanced integration
Tip: GitLens extension provides powerful Git features like:
  • Inline blame annotations showing who changed each line
  • File history and comparison tools
  • Branch comparison and management

Set Up Terminal Integration

Step 7: Open the integrated terminal

Press Ctrl+` (backtick) or go to Terminal > New Terminal

The terminal automatically opens in your project's root directory, making it easy to run Git commands, build scripts, or tests.

Explore the Project Structure

Step 8: Familiarize yourself with the codebase

Use the Explorer sidebar to navigate through:

  • com_ra_tools/ - Main component directory
  • mod_ra_tools/ - Module directory
  • README.md - Project overview
  • JOOMLA_DEVELOPMENT_SETUP.md - Development setup guide
Note: The ra-tools project is a Joomla component. Understanding Joomla's directory structure will help you navigate the code more effectively.

Quick Tips for Productivity

  • Use Ctrl+P / Cmd+P to quickly open files by name
  • Use Ctrl+Shift+F / Cmd+Shift+F to search across all files
  • Use F12 to go to definition of a function or class
  • Use Alt+Up/Down to move lines of code up or down
  • Use multi-cursor editing with Ctrl+D / Cmd+D to select next occurrence

What's Next?

With VS Code configured, you're ready to:

  • Download additional code for development
  • Create a branch for your changes
  • Start making contributions to the project
